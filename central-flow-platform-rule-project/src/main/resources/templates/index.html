限制回单规则网关

100％确定性的规则检测，无性能影响，在处理检测中可限制用户提交表单，检测后流转到“质检后驳回网关”.

质检后驳回规则网关

100％确定性的规则检测，且规则有性能影响的可能。规则执行结果为系统直接驳回该张工单，流转到“人工质检网关”.

进入人工规则质检网关

系统无法100％确定判断结果，执行结果可驳回工单，也可以进入人工复检网关，同时也可以选择进入归档环节.

进入人工复检规则网关
